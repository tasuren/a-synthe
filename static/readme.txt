◇ 概要 ◇
このソフトはマイクで拾った音に含まれる音程を検出して表示するソフトです。  
あまり便利ではないMIDI出力機能もあります。

◇ 動作条件 ◇
マイクはモノラルである必要があります。
なお、以下のマイク入力で動作を確認しています。
対応周波数：44100hz, サンプルフォーマット：32ビット (浮動小数)
(モノラルしか対応していない県については、いつかバージョンアップでステレオに対応したいと思っています。)

◇ ファイル構成 ◇
  ● Windows
aSynthe.exe: 実行ファイル (本体)
readme.txt: このファイル
static/notes.csv: 音程の番号と周波数が書いてあるファイル
static/main.ico: アイコン
licenses-windows.html: 使用したライブラリのライセンスを書き込んだHTMLファイル

  ● Mac
aSynthe.app: 実行ファイル (本体)
readme.txt: このファイル
licenses-mac.html: 使用したライブラリのライセンスを書き込んだHTMLファイル

◇ インストール ◇
  ● Windows
zipファイルからこのファイルや実行ファイルを展開するだけです。

  ● Mac
アプリケーションフォルダに`aSynthe.app`を移動することで、Launchpad(アプリ一覧)に本ソフトが追加されます。
(反映が数秒遅れるので注意。)

◇ アンインストール ◇
  ● Windows
インストール時にできたファイルを消すだけです。

  ● Mac
アプリケーションフォルダにある`aSynthe.app`を消せば良いです。

◇ つかいかた ◇
起動するとウィンドウが表示されます。
左にある`Note`というグループに音階が表示されます。
右側に色々設定するためのものが表示されます。

  ● 窓関数
内部で音階を検出するために行うフーリエ変換で使うデータを使う前に窓関数に渡すかどうか。
使うと少し精度が向上するかもしれません。

  ● 無音データを設定する
このボタンを押した時にマイクで拾った音声を無音時の音声と記憶して、それ以降の音声からその無音時の音声のデータを引きます。
これは、電子天秤のゼロ合わせのようなもので、これを使えば検出に使う音から雑音を消すことができます。

  ● 検出対象になる最低音量
音階検出の対象となる音声の音量です。
デフォルトは音声の62%(約-30db?)です。
100%は-80dbです。

  ● ポイント数をデータの長さの何倍にするか
フーリエ変換でのポイント数をフーリエ変換するデータの長さの何倍にするかです。
数値を高くすると精度が上がりますが重くなります。
デフォルトは9で、よくわからない人はいじらないで良いです。

  ● 音程調整
表示する音程を実際の音程から何オクターブずらすかです。
例えば、男性の声では周波数が低いために、とても低い音程が表示されるのを、24と入力ですることで恐らく防げます。
マイナスの値にも対応しています。

  ● MIDI出力
検出した音の一番大きい音でMIDI出力を行うかどうかです。
MIDI出力を使用可能な場合のみこのチェックボックスは有効になります。
なお、あまり使い物になりません。

◇ 上手なつかいかた ◇
小さくて音の高さが不安定な音は全く拾ってくれません。
また、マイクを口に近づけて声を出した場合、風のような音を拾ってしまい正しい結果をだしません。
録音してみるとわかりますが、本当に風の音しか聞こえません。
上手に音を拾わせるには、マイクをある程度の距離だけ離して、軽く合唱をしてるようなはりのある大きい声をだしましょう。
なお、声ではない鼻歌でも一応音を拾ってくれます。
言葉を入れるとあまり拾ってくれません。
