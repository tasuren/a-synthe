◇ 概要 ◇
このソフトは、マイクで拾った音声に含まれる音階を表示するソフトです。
あまり使い物になりませんが、一応MIDI出力もできます。
仕組みは高速フーリエ変換（FFT）を使ったものです。

◇ 動作条件 ◇
マイクはモノラルである必要があります。（できればいつか対応します。）
なお、次のマイク入力で動作を確認しています：
  - 対応周波数：44100hz
  - サンプルフォーマット：32ビット（浮動小数）

◇ インストール ◇
  ● Windows
zipファイルからこのファイルや実行ファイルを展開するだけです。

  ● Mac
アプリケーションフォルダに`aSynthe.app`を移動することで、Launchpad（アプリ一覧）にこのソフトが追加されます。
※反映が数秒遅れることに注意。

◇ アンインストール ◇
  ● Windows
インストール時にできたファイルを消すだけです。

  ● Mac
アプリケーションフォルダにある`aSynthe.app`を消せば良いです。

◇ つかいかた ◇
起動すると左側に検出した音階が表示されます。

  ● 窓関数（ハン窓）
FFTをする前にサンプルデータをいい感じに整えます。これは精度向上に繋がる可能性があります。
理屈に関して、具体的には次のウェブページをご確認ください：https://www.logical-arts.jp/archives/124

  ● 無音データを設定する
無音データのサンプルを採取し、そのサンプルの音声データを除外するようにします。
これは、雑音を無音データとして採取することで、雑音を少しなくすことができるかもしれません。
いわゆる、電子天秤のゼロ点調整のようなものです。

  ● 検出対象とする最低音量
音階の検出を試みるのに必要な最低音量です。

  ● ポイント数をデータの長さの何倍にするか
簡単にいうと、計算する量を増やします。
数値を上げれば精度が上がりますが、その分処理が大変になります。
詳細は次のウェブページをご確認してください：https://www.logical-arts.jp/archives/112

  ● 音階調整
表示する音階を実際の音階から何オクターブずらすかです。
例えば、男性の声では周波数が低いためにとても低い音階が表示されることがあり、それは24と入力ですることで恐らくいい感じにできます。
マイナスの値にも対応しています。

  ● MIDI出力先
検出した音の一番大きい音をMIDIで出力する場合の出力先です。
MIDI出力を使用可能な場合のみこの選択ボックスは有効になります。
なお、少ししか使い物になりません。

◇ 上手なつかいかた ◇
小さくて音の高さが不安定な音は全く拾ってくれません。一貫性のある音は拾われやすいです。
そしてマイクを口に近づけて声を出した場合、風を切る音にかきけされて正しく検出しません。
そこに注意しましょう。
上手に音を拾わせるには、マイクをある程度の距離だけ離し、軽く合唱のようにはりのある大きい声をだしましょう。
なお、声ではない口笛でも一応音を拾ってくれるらしいです。（残念なことに開発者は口笛をできません...）
